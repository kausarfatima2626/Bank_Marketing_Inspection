# Bank Marketing Data Analysis 🏦📞

An Exploratory Data Analysis (EDA) project focused on understanding customer behavior during banking marketing campaigns. The main goal is to analyze customer demographics and contact details to identify patterns and predict whether a client will subscribe to a term deposit.

---

## 🎯 Project Overview
This project analyzes the `bankmarketing.csv` dataset, which contains real-world data from a bank's direct marketing campaigns (phone calls). Financial institutions invest heavily in marketing; this analysis sets the foundation for optimizing those campaigns and targeting the right customers.

### Key Objectives:
* Explore customer demographics (age, job, education) and campaign touchpoints.
* Understand the factors influencing a customer's decision to subscribe to a term deposit.
* Prepare the dataset's foundations for future machine learning workflows.

---

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
* **Environment:** Jupyter Notebook
